## Guide
### Frontend
- Install nodejs version 16. : https://nodejs.org/dist/v16.2.0/node-v16.2.0-x64.msi
- Script: npm install -> npm start
### Backend
- Setup enviroment: pip install flask pandas flask_cors
- Script: flask run --port 5001
